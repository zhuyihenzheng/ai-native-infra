# 禁止事项（模板）

> 最高优先级硬规则。`/align-draft` 按本项目补充项目特有禁忌。AI 违反即视为错误产出。

## 通用（默认适用）
- 不在无关改动中改：路由路径、字段/属性名、SQL 列名、API key、表结构。
- 不引入项目未使用的技术栈 / 不擅自新增分层（Service/UseCase/Context/Setupper 等）——**贴合现状，不"现代化"遗留结构**。
- 不跳层（见架构规约的请求流）。
- 不把原始 JSON / 默认 `toString()` 直接送到视图。
- 不硬编码 code 名称（有 code master / list master 时取之）。
- 不在 fixture 写真实个人信息 / 凭证 / 客户数据 / 真实内部标识符。
- MyBatis 不用 `${}` 拼接（白名单排序列除外）。
